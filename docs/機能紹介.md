# 機能紹介（doboku-14w-training）

本アプリは、1級土木施工管理技士（第一次検定）に向けて、過去問演習を 14週間で進めるための学習支援 Webアプリ（GAS）です。

## 主な機能
- ミニテスト
  - 週ごとのテストを提示（`TestPlan14`）
  - 中断・再開（進行中 Attempt を保持）
  - 制限時間・出題数は `Config` で変更可能
- 弱点復習
  - TagStats から弱点タグ（TOP3）を算出して演習
- 進捗可視化
  - 提出済みテスト数 / 平均点 / 直近7日 / 週次推移
- 管理（Admin）
  - QuestionBank を CSV で Dry-run -> Import
  - 画像アップロード（QuestionBank `imageUrl` 更新）

## データ構造（Sheets）
- `Config`
- `Users` / `UserAccess`
- `QuestionBank`
- `TestPlan14` / `TestSets`
- `Attempts` / `AttemptAnswers`
- `TagStats`

## 変更ポイント（archi との差分）
- ConceptCards / LawCards は使用しません。
- localStorage のキーは `doboku14w_` で名前空間分離しています。
