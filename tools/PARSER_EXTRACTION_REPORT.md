## 🔧 技術メモ

### パーサーパターン

#### 旧形式（R1-R3）
```regex
r'([１２３４])→(.+?)(?=[１２３４]→|$)'
```

#### 中間形式（R4-R5）
```regex
r'選択肢(\d+)\.\s*(.+?)(?=選択肢\d+\.|$)'
```

#### 新形式（R7）
```python
# 【解説】セクション全体を統合解説として取得
# 選択肢別パターンなし
```
