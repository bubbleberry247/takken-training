## AI生成プロンプト例（参考）

### 基本形
```
以下の1級土木施工管理技士の問題について、各選択肢A-Dの説明文を生成してください。

**問題文**: {stem}
**選択肢A**: {choiceA}
**選択肢B**: {choiceB}
**選択肢C**: {choiceC}
**選択肢D**: {choiceD}
**正解**: {correct}

**説明文の要件**:
- 各選択肢が正しい/誤りである理由を簡潔に説明
- 関連する法令・基準があれば引用
- 初学者にも理解できる平易な日本語
- 各説明は50-150文字程度

**出力形式**:
explainA: <説明A>
explainB: <説明B>
explainC: <説明C>
explainD: <説明D>
```

### 既存説明を参考にする場合
```
以下の既存の説明を参考に、同じトーン・長さで説明文を生成してください。

**参考例（R4gakkaB-001）**:
explainA: 労働安全衛生法第30条により、同一場所で複数の事業者が作業を行う場合、元方事業者が統括安全衛生管理を実施する必要がある。正しい。
explainB: ...

**新規生成対象**:
問題文: {stem}
...
```

## レビュー基準

### 必須チェック項目
- [ ] 説明文が選択肢の正誤を正しく説明している
- [ ] 専門用語の誤りがない
- [ ] 文法・誤字脱字がない
- [ ] 長さが適切（50-150文字）

### 望ましいチェック項目
- [ ] 関連法令・基準が引用されている
- [ ] 初学者にも理解できる
- [ ] 他の選択肢の説明と重複していない
