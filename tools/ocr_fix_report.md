## カテゴリ3: 表記ゆれ (63件)【一括置換推奨】

### 「締め固め」→「締固め」

土木用語として正式な表記は**「締固め」**(送り仮名なし)。
JIS規格、土木学会用語集、国土交通省の仕様書でも「締固め」が標準。

**検出数**: 63件
**確信度**: 低(表記ゆれのため意味は正しい)
**推奨対応**: 全置換

**該当行番号(抜粋)**:
- 2 (R1gakkaA-001, choiceC)
- 3 (R1gakkaA-002, choiceB)
- 24 (R1gakkaA-023, choiceC)
- 30 (R1gakkaA-029, choiceB, choiceC)
- ... 他58件

**一括修正スクリプト**:
```python
import csv

# 全カラムで「締め固め」→「締固め」
text_columns = ['stem', 'choiceA', 'choiceB', 'choiceC', 'choiceD', 'choiceE',
                'explainA', 'explainB', 'explainC', 'explainD', 'explainE',
                'explainShort', 'explainLong']

for row in rows:
    for col in text_columns:
        if row[col]:
            row[col] = row[col].replace('締め固め', '締固め')
```
