# 選択肢別説明 手動追記テンプレート

## 使い方

1. `split_failed_cases_csv.json` から補完したい問題を選ぶ
2. 原本データ（Wordファイルまたは過去問サイト）で該当問題を確認
3. このテンプレートに従って選択肢別説明を作成
4. `questionbank_drive_import.csv` の該当行に追記

---

## テンプレート形式

### 判定ワード

- **正解の場合**: 適当。/ 正しいです。/ 設問の通りです。
- **不正解の場合**: 不適当。/ 誤りです。/ 適当ではありません。

### 説明構成

```
[判定]。[選択肢の内容の要約または補足]。[理由・根拠・追加情報]。
```

## 記入例

### 例1: R2gakkaA-006 コンクリート用骨材

**問題文**: コンクリート用骨材に関する次の記述のうち、適当でないものはどれか。

**選択肢**:
1. 細骨材は、10mm のふるいをすべて通過し、5mmのふるいをすべて通過するか、又は、質量で85%以上通過するものをいう。
2. 粗骨材の粘土塊量は、質量で1.0%以下のものでなければならない。
3. 細骨材の有機不純物は、標準色液の色より淡くなければならない。
4. 粗骨材の最大寸法は、鉄筋のあきの4/5 以下でなければならない。

**正解**: 2

**統合解説（既存）**: 不適当なものは２です。細骨材の粘土塊量は1.0％以下のものを用いなければなりません。また、粗骨材の粘土塊量は0.25％以下です。

---

**追記内容**:

```csv
explainA,適当。細骨材は10mmのふるいをすべて通過し、5mmのふるいを85%以上通過するものを指します。
explainB,不適当。粗骨材の粘土塊量は0.25%以下でなければなりません。1.0%という記述は細骨材の基準です。
explainC,適当。細骨材の有機不純物は標準色液より淡い色でなければなりません。有機物含有量の制限です。
explainD,適当。粗骨材の最大寸法は鉄筋のあきの4/5以下です。これによりコンクリートが鉄筋間に十分充填されます。
```

### 例2: R2gakkaA-007 混和材料

**問題文**: コンクリート用混和材料に関する次の記述のうち、適当でないものはどれか。

**選択肢**:
1. 高性能AE減水剤は、所要のワーカビリティーを得るための単位水量を減少させ、コンクリートの強度を増加させる効果がある。
2. フライアッシュを用いると、コンクリートの水和熱が増加し、コンクリートの温度上昇が大きくなる。
3. 膨張材を用いると、コンクリートのひび割れを抑制する効果がある。
4. AE剤を用いると、コンクリートの耐凍害性が向上する。

**正解**: 2

**統合解説（既存）**: 不適当なものは２です。フライアッシュを用いると、コンクリートの水和熱が低減し、コンクリートの温度上昇を抑えることができます。温度上昇の増加ではありません。

---

**追記内容**:

```csv
explainA,適当。高性能AE減水剤は単位水量を減少させ、コンクリート強度を増加させる効果があります。
explainB,不適当。フライアッシュは水和熱を低減し、温度上昇を抑える効果があります。温度上昇が増加するという記述は誤りです。
explainC,適当。膨張材はコンクリートのひび割れを抑制する効果があります。
explainD,適当。AE剤は微細な気泡を連行し、コンクリートの耐凍害性を向上させます。
```

## 品質チェックリスト

追記前に以下を確認してください:

- [ ] 全ての選択肢（A-D）に説明を記載した
- [ ] 判定ワード（適当/不適当）を明記した
- [ ] 不正解選択肢には「なぜ誤りか」を説明した
- [ ] 正解選択肢には補足情報を追加した（可能な場合）
- [ ] 原本データまたは信頼できる情報源を参照した
- [ ] 推測や憶測ではなく、事実に基づいた説明にした
- [ ] 文体を統一した（「です・ます調」または「である調」）

## CSV追記手順

### 手順1: CSVファイルをバックアップ

```bash
cp questionbank_drive_import.csv questionbank_drive_import_backup_$(date +%Y%m%d_%H%M%S).csv
```

### 手順2: 該当行を編集

ExcelまたはGoogle Sheetsで該当行の `explainA`, `explainB`, `explainC`, `explainD` 列に追記

### 手順3: 保存してコミット

```bash
git add questionbank_drive_import.csv
git commit -m "追加: R2gakkaA-XXX の選択肢別説明"
```

## よくある質問

### Q1: 統合解説に「設問の通りです」としか書かれていない場合は?

**A**: 選択肢の内容を簡潔に要約してください。

例:
```
選択肢: 盛土の締固め管理では、締固め機械の走行位置を追尾・記録することで、規定の締固め度が得られる締固め回数の管理を厳密に行うことができます。
統合解説: 適当。設問の通りです。

→ explainD: 適当。締固め機械の走行位置を追尾・記録することで、締固め回数を厳密に管理できます。
```

### Q2: 不正解選択肢の理由が原本にない場合は?

**A**: 正解選択肢の説明から逆算できる場合のみ記載してください。推測は避けてください。

例:
```
統合解説: 不適当なものは２です。粗骨材の粘土塊量は0.25％以下です。

→ explainB: 不適当。粗骨材の粘土塊量は0.25%以下でなければなりません。1.0%という記述は誤りです。
```

### Q3: 専門用語の補足説明はどこまで必要?

**A**: 基本的には不要です。ただし、学習者が理解しやすくなる補足があれば簡潔に追加してください。

例:
```
explainD: 適当。AE剤は微細な気泡を連行し（エントレインドエア）、コンクリートの耐凍害性を向上させます。
```

## 参考: 良い説明の例（既存データより）

### R2gakkaA-001 選択肢1

```
explainA: 誤りです。RI計器による土の密度試験では、土の含水比のほかに乾燥密度や湿潤密度を求めることができ、盛土等の締固め度の判定等が行えますが、地盤の許容支持力を算定することはできません。地盤の許容支持力を求めるには、標準貫入試験を行い、N値を求めることにより算定できます。
```

**良い点**:
- ✅ 選択肢のどこが誤りかを明確に指摘
- ✅ 正しい情報（標準貫入試験）を補足
- ✅ 理由を具体的に説明

### R2gakkaA-003 選択肢2

```
explainB: 正しいです。マシンガイダンス技術は、TS（トータルステーション）やGNSS（衛星測位システム）の計測技術を用いて、施工機械の位置情報・施工情報及び施工状況と三次元設計データとの差分をオペレータに提供する技術です。
```

**良い点**:
- ✅ 技術の定義を明確に説明
- ✅ 略語の正式名称を併記（TS、GNSS）
- ✅ 具体的な機能を記載

## まとめ

このテンプレートを活用して、ユーザーからフィードバックがあった問題や、特に重要な問題から順次説明を補完していきましょう。

**目標**: 需要の高い問題から段階的に改善し、学習者の満足度を向上させる。
